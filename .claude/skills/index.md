# Todo Application Skills Index

This document provides an overview of all the skills defined for the TodoPythonSubAgent to implement the core features of the Python Todo Application.

## Core Skills

-   **[Add Task](./add_task/add_task.md)**
    -   *Description*: Guides the implementation of the `add <description>` command to create a new task.

-   **[View Tasks](./view_tasks/view_tasks.md)**
    -   *Description*: Guides the implementation of the `list` command to display all tasks.

-   **[Mark Task as Complete](./mark_complete/mark_complete.md)**
    -   *Description*: Guides the implementation of the `done <task_id>` command to mark a task as completed.

-   **[Delete Task](./delete_task/delete_task.md)**
    -   *Description*: Guides the implementation of the `delete <task_id>` command to remove a task from the list.
